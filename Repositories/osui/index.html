What is OSUI?
    OSUI is a desktop environment and a web interface for the Smith-Pad ecosystem
    Available on the [experiment-flask] branch. This version is not highly recommended for use in production. The recommended version to run is between the development or stable branches of OSUI
      Will be available for recommended use in production soon. https://github.com/smith-pad/osui/tree/experiment-flask
        After the merging to the development and stable branches, the [experiment-flask] branch will be deleted.
